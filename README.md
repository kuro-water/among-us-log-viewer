# 🎮 Among Us ゲーム履歴ビューワー

シンプルで高速なゲーム履歴分析ダッシュボード。インストール不要、ブラウザだけで動作します。<br>
⚠️ **KTR（身内）向けなので、一般的な among us 環境では動作しません。**

## ✨ 主な機能

### 📊 データ分析

-   **プレイヤー別勝率グラフ** - TOP15 プレイヤーの勝率を棒グラフで可視化
-   **マップ別統計** - プレイされたマップごとのゲーム数を表示
-   **プレイヤー統計** - 全プレイヤーの詳細な勝率、生存率、ロール別勝率を表示（カスタムロール対応）
-   **時系列ゲーム結果** - トッププレイヤーの累積勝利数を時間軸で表示
-   **ランキング** - 勝利数ランキングと勝率ランキング（3 ゲーム以上）
-   **ロール別勝率分析** - すべてのロール別の勝率を分析（MOD カスタムロール対応）
-   **ゲーム一覧** - 全ゲームの詳細情報を表形式で表示

### 🎨 UI 特性

-   ✅ **ダークテーマ** - Among Us のビジュアルに合わせた赤 × 水色配色
-   ✅ **レスポンシブデザイン** - スマートフォンからタブレット、デスクトップまで対応
-   ✅ **複数ファイル構成** - HTML、CSS、JavaScript を分割し保守性向上
-   ✅ **オフライン対応** - インターネット接続不要（CDN は最初の読み込みのみ）
-   ✅ **MOD 対応** - カスタムロール自動検出・表示

## 🚀 使用方法

### 1. ファイルの準備

```
game_history.jsonl ファイルを用意します
（MOD作成者から提供されるログファイル）
```

### 2. ビューワーを開く

以下のいずれかの方法で `index.html` をブラウザで開きます：

-   **方法 A（最シンプル）**: ファイルエクスプローラーから `index.html` をダブルクリック
-   **方法 B（推奨）**: ターミナルでローカルサーバーを起動

```bash
# Python 3
python -m http.server 8000

# または Node.js
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセス

### ファイル読み込み方法

ブラウザで `index.html` を開いた後：

1. **方法 A（ドラッグ&ドロップ）**: `game_history.jsonl` を「📤 アップロードゾーン」にドラッグ&ドロップ
2. **方法 B（ファイル選択）**: 「ファイルを選択」ボタンをクリックして `game_history.jsonl` を選択

ファイルが正常に読み込まれると、ダッシュボードが自動的に表示されます。
ファイル読み込み後、以下のタブで分析を確認できます：

| タブ              | 説明                           |
| ----------------- | ------------------------------ |
| 📊 概要           | 勝率分布とマップ別統計         |
| 🎮 プレイヤー統計 | 全プレイヤーの詳細成績表       |
| 📈 時系列         | ゲーム数の増加に伴う勝利数推移 |
| 🏆 ランキング     | 勝利数と勝率のランキング       |
| 🎯 ロール分析     | 役職別の勝率比較               |
| 📋 ゲーム一覧     | 全ゲームのログ表               |

## ⚙️ トラブルシューティング

### ファイルが読み込めない場合

-   `.jsonl` ファイルか確認してください
-   JSON フォーマットが正しいか確認してください
-   ブラウザの開発者ツール（F12）のコンソールでエラーを確認

### グラフが表示されない場合

-   ブラウザをリロード（Ctrl+R または ⌘+R）
-   JavaScript が有効か確認
-   CDN（Chart.js、Bootstrap）に接続できるか確認
